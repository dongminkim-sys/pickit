# 핔잇 크롬 확장프로그램

Meta Ad Library, 네이버 GFA 광고를 핔잇에 바로 수집할 수 있는 크롬 확장프로그램입니다.

## 설치

터미널에 아래 명령어를 붙여넣고 실행하세요.

```bash
bash <(curl -s https://raw.githubusercontent.com/dongminkim-sys/pickit/main/chrome-extension/install.sh)
```

스크립트가 자동으로:
- 확장프로그램 파일을 다운로드합니다 (`~/pickit-extension/`)
- 폴더 경로를 클립보드에 복사합니다
- 크롬 확장프로그램 페이지를 엽니다

이후 크롬에서:
1. 우측 상단 **개발자 모드** 토글을 켭니다
2. **압축 해제된 확장 프로그램을 로드합니다** 클릭
3. 클립보드에 복사된 경로를 붙여넣기하여 폴더를 선택합니다

## 초기 설정

설치 후 확장프로그램 아이콘(퍼즐 모양)을 클릭하면 설정 팝업이 열립니다.

| 항목 | 설정값 |
|------|--------|
| 서버 URL | `https://pickit.to` |
| 사용자 | 본인 이름 선택 |
| 기본 카테고리 | 자주 사용하는 카테고리 (선택) |

## 사용법

### Meta Ad Library에서 수집

1. [Meta Ad Library](https://www.facebook.com/ads/library)에 접속
2. 원하는 광고를 검색
3. 광고 카드에 나타나는 **핔잇!** 버튼 클릭
4. 사이드 패널에서 정보 확인/수정 후 **수집하기**

### 네이버 GFA 광고 수집

1. 네이버, 네이버 뉴스, 블로그 등에 접속
2. 광고 영역(AD/광고 표시)에 나타나는 **핔잇!** 버튼 클릭
3. 사이드 패널에서 정보 확인/수정 후 **수집하기**

## 업데이트

같은 설치 명령어를 다시 실행하면 최신 버전으로 업데이트됩니다.

```bash
bash <(curl -s https://raw.githubusercontent.com/dongminkim-sys/pickit/main/chrome-extension/install.sh)
```

실행 후 크롬에서 `chrome://extensions` → 핔잇 카드의 **새로고침 버튼(↻)** 을 클릭하세요.

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 핔잇! 버튼이 안 보여요 | 페이지 새로고침 후 다시 확인 |
| 수집 버튼 눌러도 반응이 없어요 | 팝업에서 서버 URL이 `https://pickit.to`인지 확인 |
| "서버에 연결할 수 없습니다" | 핔잇 서버가 실행 중인지 확인 |
| 이미 수집한 광고인데 버튼이 또 나와요 | 크롬 확장프로그램 새로고침(↻) 후 페이지 새로고침 |
